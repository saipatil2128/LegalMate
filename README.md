# LegalMate

LegalMate is a full-stack web application designed to simplify Indian legal procedures. It provides step-by-step guidance for case filing, legal workflows, and access to ready-to-use contract templates.

---

## Features
- User Registration & Login System  
- Guided Case Filing Process  
- Step-by-Step Legal Procedure Navigation  
- Ready-to-use Contract Templates  
- User-friendly Interface  

---

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js  
- Database: MySQL  

---

## Objective
To make legal processes easy, accessible, and understandable for common users without requiring extensive legal knowledge.
